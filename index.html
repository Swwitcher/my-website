02.Ma formation

J'ai commencé l'informatique au début de ma licence en 2017 à l'Université de Bordeaux avec du python pour découvrir les bases. Nous avons ensuite découvert le C, langage avec le quel j'ai pu apprendre l'algorithmique. Viens ensuite lors de ma L3 ma rencontre avec la programmation orientée objet, et plus précisement le Java ! À travers différents cours, j'apprends le DDD et les design patterns.

Vient la première année de master, où je choisis la spécialisation Génie Logiciel. Master durant le quel j'ai pu apprendre la programmation parallèle, le big data, la programmation web, mais aussi durant le quel j'ai effectué des projets de groupe et appris à les gérer à l'aide de plusieurs méthodes et de différentes manières.

03.Mes projets perso

Chess
À la recherche d'un site d'échecs où l'on peut jouer à deux sur le même ordinateur ? Vous l'avez touvé ! Il manque pour l'instant seulement l'obligation de bouger lorsque l'on est en échec et la détection d'échec et mat.
  • Javascript
  • PhaserJS
Pong
Un pong classique à deux joueurs, l'un avec les flèches, et l'autre avec Q - D. J'ai eu l'idée de ce site pour découvrir les possibilités de PhaserJS.
  • Javascript
  • PhaserJS
SplitBros
Un jeu crée avec Unity qui consiste à controler en même temps deux personnages qui n'ont pas le même terrain. Ce jeu m'a permis de découvrir Unity, le déplacement d'unités, les collisions et la géstion des scènes.
  • Javascript
  • PhaserJS
DVD
Nostalgique de l'écran de veille des lecteurs DVD ? Ce site vous rappelera des souvenirs ! Ce site a été dans la continuité de ma découverte de PhaserJS. Ici j'ai pu expérimenter la gestion du côté responsive.
  • Javascript
  • PhaserJS
NFT preview
Challenge trouvé sur Frontend Mentor réalisé en moins d'une heure. Réalisé pour tester les challenges et apprendre à se concentrer sur les elements importants grâce à la limite de temps que je me suis imposée.
  • HTML/CSS

04.Mes projets universitaires

Blackjack
Un site de Blackjack qui consistait à travailler sur la partie HTML/CSS ainsi que la gestion du DOM et des boutons avec le Javascript. Il n'y a pas de détection pour savoir si l'on a gagné, donc pas encore de gestion des jetons.
  • HTML/CSS
  • Javascript
Meteo
Site qui affiche des données météo récupérées sur un autre site. Le but est donc de récupérer un nom de ville soit à l'aide d'un formulaire, soit en récupérant un click sur la carte. Puis d'en afficher la météo du moment ou d'un autre jour.
  • HTML/CSS
  • Javascript
PostFlex
Le but ici était de travailler en mode agile et gérer le contact avec le client. Il fallait produire un site équivalent à Trello en y ajoutant la possibilité de gérer son projet de manière agile. Cela m'a permis de découvrir React.
  • HTML/CSS
  • React
QRcode dynamique
La demande du client était de pouvoir afficher sur un site un QRcode qui change en fonction de l'heure. Pour cela nous générons le QRcode avec python et l'affichage est effectué en Javascript et PHP pour la redirection.
  • Python
  • PHP
  • Javascript
Plant&Moi
Site où l'on travaille avec une API qui nous permet de donner le nom d'une plante en lui donnant une image. Par la suite il sera possible de se connecter pour publier ses photos de plantes.
Site encore en développement.
  • HTML/CSS
  • React
Age of Empire like
Jeu de gestion d'unités où l'on doit capturer des chateaux contre un ordinateur. Application développée afin de prendre en main le DDD, JavaFX pour l'affichage, ainsi que le développement de tests unitaires.
  • Java
  • JavaFX
Robot traqueur
Projet qui consistait à faire suivre un robot toujours à 1m d'une cible. Pour cela le robot dispose de 2 cameras dont on récupère les images et les traîte avec OpenMP afin de connaître la distance de la cible. Simulé sous Unity.
  • OpenMP
  • Unity3D
